Privacy Policy

Last updated: 2026-04-09

Govee Light Management ("the plugin") is a Stream Deck plugin developed by Felix Geelhaar. This privacy policy explains what information the plugin handles and how.

Summary

The plugin does not collect, transmit, or store any personal data on third-party servers. It communicates only with the official Govee API, using a credential that you provide and that is stored locally on your device.

Information the plugin handles

  • Govee API key. You generate this yourself at developer.govee.com and enter it into the plugin's Property Inspector. The key is stored in Stream Deck's encrypted settings on your local machine. It is never transmitted anywhere except directly to Govee's official API endpoints (openapi.api.govee.com) to send light-control commands.
  • Device information. Device IDs, names, and model numbers returned by the Govee API are cached in memory for short periods to improve responsiveness. This information is never sent to any server other than Govee's API.
  • Light configuration. Action-specific settings (selected device, brightness presets, colors, etc.) are stored in Stream Deck's local settings storage and never leave your device.
  • Diagnostic metrics. The plugin keeps in-memory telemetry (request counts, latency) used only by the local diagnostics panel. Nothing is transmitted off-device.

What the plugin does NOT do

  • It does not collect personal information, names, emails, or location data.
  • It does not use analytics, tracking, crash reporters, or telemetry services.
  • It does not contact any server other than the official Govee API.
  • It does not share or sell any data to third parties.
  • It does not use cookies or tracking technologies.

Third-party services

The plugin communicates with the Govee Developer API to control your lights. Govee's own privacy policy governs how they handle requests made to their API. Please review it at govee.com/privacy-policy.

Data retention and deletion

All data handled by the plugin (API key, device settings, diagnostics) lives only on your local device. To delete it, uninstall the plugin from Stream Deck or remove the action from your button layout. You can also revoke your Govee API key at any time in the Govee app.

Children's privacy

The plugin is not directed at children under 13 and does not knowingly collect any data from them.

Changes to this policy

If this policy changes, the updated version will be published at this URL along with a new "Last updated" date. Material changes will also be noted in the plugin's GitHub release notes.

Contact

Questions or concerns? Open an issue at github.com/felixgeelhaar/govee-light-management/issues.
