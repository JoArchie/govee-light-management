Govee Light Management

Control your Govee smart lights directly from your Stream Deck.

Get on Marketplace View on GitHub

Features

On / Off

Toggle individual lights or entire groups instantly with a single tap.

Brightness

Set precise brightness levels from a keypad button or adjust smoothly with a Stream Deck+ dial.

Color

Pick any color from the full spectrum. Dial rotation scrubs through hues with live feedback.

Color Temperature

Shift between warm and cool white (2000K to 9000K) to match any mood.

Segment Control

For RGB IC lights: apply rainbow, gradient, or solid presets across all segments, or tune individual segments.

Stream Deck+ Dials

Tactile dial controls with custom gradient touchscreen feedback. Rotate to adjust, press to toggle.

Setup

  1. Install the plugin from the Elgato Marketplace.
  2. Get a Govee API key at developer.govee.com.
  3. Add an action to your Stream Deck and paste the API key in the Property Inspector.
  4. Select a light from the dropdown and you're done.

Your API key is stored securely in Stream Deck's encrypted settings and never leaves your machine except to communicate with the official Govee API.

Requirements

Stream Deck

Stream Deck software 6.9 or later. Any Stream Deck hardware; Stream Deck+ recommended for dial actions.

Operating System

macOS 12 or later, or Windows 10 or later.

Govee Account

A free Govee developer API key and at least one supported Govee smart light.
